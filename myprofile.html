Intelligent Virtual Queue System

Intelligent Virtual Queue System is a web-based application built to improve queue management in environments like hospitals and banks. It allows users to register, log in securely, and book service slots online, reducing physical wait times and improving user experience.

The system includes a personal dashboard where users can view their bookings, and it ensures data integrity with unique email validation and time-slot conflict prevention.

Key Features: Online slot booking for hospitals and banks, real-time queue tracking, unique email validation, personalized dashboard with booking history, and Hibernate-powered data persistence.

Tech Stack: Java, Servlets, JSP, PostgreSQL, Hibernate (JPA), HTML, CSS, Apache Tomcat, Maven
